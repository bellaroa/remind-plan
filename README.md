## 프로젝트 명

Remind Plan

## 개요

Remind Plan은 일정을 계획하고, 그것을 꾸준히 리마인드해준다는 의미로, 사용자가 할 일을 잊지 않도록 알림을 필수로 등록하고 완료할 수 있도록 도와줍니다. 목표에 대한 의지를 반영하며, 하루 내로 계획한 일을 효과적으로 관리할 수 있는 간단한 인터페이스를 제공합니다.

## 기간
